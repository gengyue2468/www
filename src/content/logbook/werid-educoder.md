前几天做头歌的数据结构实验，检查点总是过不了，提示:

```cpp
湖北 1 2 3 4 5╕↵
湖南 6 7 8 9 10╕↵
```

一开始还以为是 `\n` 换行符的锅，折腾了半天发现输出是评测程序决定的，怎么改前面的程序都不太行。

晚上快断电了突发奇想，既然评测程序依赖  `ListTraverse` 遍历线性表拿结果，不会是是最后不允许有空格吧。

还真是，把空格去掉就行了，真纯傻逼吧...

```cpp
for(int i = 0; i < L.length; i++){
    if (i == L.length - 1) printf("%d", L.elem[i]);
    else printf("%d ", L.elem[i]);
}
```
